#  Data Sources

This project relies on continuous data streaming and historical records from multiple public sources.

## I. U.S. Geological Survey (USGS)

### Site: Deschutes River Near Madras, OR (USGS-14092500)

* **Primary Data Link (Gage Height):**
    [https://waterdata.usgs.gov/monitoring-location/USGS-14092500/#dataTypeId=continuous-00065-0&period=P7D&showFieldMeasurements=true](https://waterdata.usgs.gov/monitoring-location/USGS-14092500/#dataTypeId=continuous-00065-0&period=P7D&showFieldMeasurements=true)

* **Data Type:** Continuous Gage Height (ft).

### Site: Crooked River Near Prineville, OR (USGS-14070000)

* **Link:** [https://waterdata.usgs.gov/monitoring-location/14070000/](https://waterdata.usgs.gov/monitoring-location/14070000/)
